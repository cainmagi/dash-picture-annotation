# Overview of APIs

<APITopBar type="package" source="." />

Dash porting version of the react project
[React Picture Annotation<IconExternalLink/>][git-react-picture-annotation]. Provide
a simple annotation window for a single picture.

This graph shows the high-level overview of the whole package. The core component is
`DashPictureAnnotation`. Some typehints and utilities are provided for handling the
annotation data.
