## 关键字 {#keywords}

|  关键字  |   类型   |               必选                | <center>说明</center>                          |
| :------: | :------: | :-------------------------------: | :--------------------------------------------- |
|   `x`    | `float`  | <InlineIcon icon={vsiCheckIcon}/> | 标记左上角位置的x轴坐标。                    |
|   `y`    | `float`  | <InlineIcon icon={vsiCheckIcon}/> | 标记左上角位置的y轴坐标。                    |
| `width`  | `float`  | <InlineIcon icon={vsiCheckIcon}/> | 边界框的宽度。                                 |
| `height` | `float`  | <InlineIcon icon={vsiCheckIcon}/> | 边界框的高度。                                 |
|  `type`  | `"RECT"` | <InlineIcon icon={vsiCheckIcon}/> | 标记的形状类型。目前，仅支持`"RECT"`（矩形）。 |
