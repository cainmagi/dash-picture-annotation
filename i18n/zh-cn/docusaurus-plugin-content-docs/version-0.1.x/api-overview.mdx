# API总览

<APITopBar type="package" source="." />

本组件包是对React项目[React Picture Annotation<IconExternalLink/>][git-react-picture-annotation]的Dash移植版，用以提供针对图片的简易标记编辑器。

</div>

上图展示了整个模块的大体结构。其中，核心功能是`DashPictureAnnotation`组件。另外，提供了一些用来处理标记数据的类型（typehints）和工具。
