`DashSelectOptionItem`是`TypedDict`。

Dash选单组件的一个选项。

用来标注标记器的可选项。其用法与选单组件`dcc.Dropdown(options=...)`的属性相似。一个选项代表的是一个可用的标记类型。

## 别名 {#aliases}

该类型可以按以下方式之一获取

```python
import dash_picture_annotation as dpa


dpa.DashSelectOptionItem
dpa.typehints.DashSelectOptionItem
```
